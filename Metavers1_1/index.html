"Gameplay"

Deux modes


SIMPLE -> utilisateur affecté par la gravité

CREATIVE -> utilisateur non-affecté par la gravité


Mode de construction


Chaque bloc a sa propre grille adapté à la taille du bloc. Par exemple, si un bloc fait 0.5 sur sa largeur, alors vous pouvez placer deux fois ce bloc sur la largeur.


Chaque bloc a sa propre grille rotative. Un bloc peut être placé selon plusieurs rotations différentes (en général 8).


Possibilité d'importer ses propres blocs à condition qu'ils soient au format .json et compatible avec la librairie three.js, pour cela dans la console, tapez Block.loadGeometryJson(path) si vous avez un fichier geometry.json, ou Block.loadObjJson(path) si vous avez un fichier 3DObject.json


Accès au monde virtuel

Pour accéder au monde virtuel, appuyez sur le bouton en haut à gauche dans la barre de navigation

Contrôles Ordinateur

( PC / Mac / MacBook )

Z Q S D : déplacements avant gauche arrière droite

SOURIS : rotations caméra

O : séléctionner le bloc suivant (par défaut : aucun bloc)

M : séléctionner le mode de jeu suivant (par défaut : 'CREATIVE')

CLIC GAUCHE : supprimer le bloc visé

CLIC DROIT : placer le bloc séléctionné à l'endroit visé

ESPACE : sauter - s'envoler/monter (SIMPLE - CREATIVE)

SHIFT : . - descendre (SIMPLE - CREATIVE)

Contrôles Mobile

( Téléphones / Tablettes )

Pour le moment, les contôles se font via boutons
Boutons 'avancer' et 'saut' en bas à droite
Boutons de rotations en bas à gauche

Ces contrôles seront ultérieurement remplacés par un joystick mobile ou éventuellement par capteur de rotations de l'appareil.
